建站过程中东抄西抄的代码整理（更新中）

2022-08-20 Views 出埃及记220字1 min read繁

Markdown语法

折叠

<details>
<summary>Title</summary>

内容（上面别忘了空一行）
</details>

很大方的空行

</br>

总嫌弃用markdown写文章没办法打很多空行吗！用这个就够了！


html（魔改）

只有一半字高的高亮

示例

Markdown代码 
<span class="half_background">示例</span>
在main.less中加入（颜色可指定） 
.half_background {
    font-weight: bold;
    background-image: linear-gradient(to top, rgba(211, 182, 124, 0.4) 50%, transparent 50%);
}

全字高的高亮

示例

Markdown代码 
<span class="all_background">示例</span>
在main.less中加入（颜色可指定） 
.all_background {
    font-weight: bold;
    background-color: rgba(211, 182, 124, 0.4);
}
以上两种在main中也可以多做几种样式并错开命名，方便按需取用。

指定字体和字重

示例：EB Garamond

<font face="EB Garamond">指定英文字体</font>
